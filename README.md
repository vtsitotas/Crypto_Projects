## Installation Instructions

To get started with these projects, follow the installation instructions below:

### Prerequisites
- **Python 3.x**
- **Node.js**

### Installation Steps
1. Clone this repository:
   ```bash
   git clone https://github.com/vtsitotas/Crypto_Projects.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Crypto_Projects
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

To run the projects, use the following commands:

- For Project A:
  ```bash
  python project_a.py
  ```
- For Project B:
  ```bash
  node project_b.js
  ```

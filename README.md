# Crypto_Projects
---

### **English Description**

**Project Title:** Cryptography & Lattice-based Crypto Projects

**Description:**
This repository contains a collection of academic projects implemented in **Python** (and **SageMath**) as part of the University Cryptography coursework. The projects cover the full spectrum of modern cryptography, ranging from symmetric ciphers and number theory to advanced lattice-based cryptanalysis.

The repository is divided into three main sections:

**1. Symmetric Cryptography & Stream Ciphers**
Implementation and analysis of fundamental algorithms:

* 
**Classical Ciphers:** Statistical analysis and decryption of substitution ciphers.


* 
**Stream Ciphers:** Implementation of **LFSR** (Linear Feedback Shift Registers) and **RC4** algorithm.


* 
**PRNGs:** Analysis of Linear Congruential Generators (LCG) and frequency histograms.


* 
**Block Ciphers:** Analysis of **AES-128** modes (ECB vs CBC) and demonstration of the **Avalanche Effect**.


* 
**S-Box Analysis:** Calculation of Differential Uniformity and Linear Branch Number (LBN).



**2. Public Key Cryptography & Number Theory**
Algorithms focusing on mathematical foundations and RSA security:

* 
**Primality Testing:** Implementation of **Fermat** and **Miller-Rabin** tests , including detection of Sophie Germain primes.


* 
**Factorization:** Implementation of **Pollard's Rho** algorithm (parallelized) and Trial Division.


* 
**RSA:** Key generation, encryption/decryption, and cryptanalysis using **Wiener’s Attack** on small private exponents.


* 
**Protocols:** Analysis of **TLS 1.2 vs TLS 1.3** and PGP/GPG usage.


* 
**Modular Arithmetic:** Fast Exponentiation and Karatsuba Multiplication.



**3. Lattice-based Cryptography**
Advanced cryptanalysis using lattice reduction techniques:

* 
**Fundamentals:** Implementation of **SVP** (Shortest Vector Problem) and **CVP** (Closest Vector Problem) solvers.


* 
**Lattice Reduction:** Implementation of **Gauss-Lagrange** (2D) and the **LLL (Lenstra-Lenstra-Lovász)** algorithm.


* 
**Cryptanalysis:** Solving **Knapsack (Subset Sum)** problems using LLL and **Coppersmith’s Method** for finding small roots.


* 
**Enumeration:** Implementation of **Babai’s Nearest Plane** and **Schnorr-Euchner** enumeration.



**Tools & Languages:**

* Python 3
* SageMath
* LaTeX (for documentation)

### **Greek Description**

**Τίτλος:** Εργασίες Κρυπτογραφίας & Πλεγμάτων (Cryptography & Lattices)

**Περιγραφή:**
Αυτό το αποθετήριο περιλαμβάνει μια συλλογή από ακαδημαϊκές εργασίες υλοποιημένες σε **Python** (και **SageMath**) στα πλαίσια του μαθήματος Κρυπτογραφίας. Οι εργασίες καλύπτουν όλο το φάσμα της σύγχρονης κρυπτογραφίας, από συμμετρικά κρυπτοσυστήματα και θεωρία αριθμών έως προχωρημένη κρυπτανάλυση με χρήση Πλεγμάτων (Lattices).

Το υλικό χωρίζεται σε τρεις κύριες ενότητες:

**1. Συμμετρική Κρυπτογραφία & Κρυπτοσυστήματα Ροής**
Υλοποίηση και ανάλυση βασικών αλγορίθμων:

* 
**Κλασική Κρυπτογραφία:** Στατιστική ανάλυση και αποκρυπτογράφηση κλασικών κωδίκων αντικατάστασης.


* 
**Stream Ciphers:** Υλοποίηση καταχωρητών ολίσθησης **LFSR** και του αλγορίθμου **RC4**.


* 
**Γεννήτριες Ψευδοτυχαίων Αριθμών:** Ανάλυση γεννητριών **LCG** και ιστογράμματα συχνοτήτων.


* 
**Block Ciphers:** Μελέτη του **AES-128** (ECB vs CBC) και επίδειξη του φαινομένου χιονοστιβάδας (**Avalanche Effect**).


* 
**Ανάλυση S-Box:** Υπολογισμός Διαφορικής Ομοιομορφίας (Differential Uniformity) και Linear Branch Number (LBN).



**2. Κρυπτογραφία Δημοσίου Κλειδιού & Θεωρία Αριθμών**
Αλγόριθμοι που εστιάζουν στο μαθηματικό υπόβαθρο και την ασφάλεια του RSA:

* 
**Έλεγχοι Πρωτογένειας:** Υλοποίηση των τεστ **Fermat** και **Miller-Rabin** , καθώς και εύρεση πρώτων Sophie Germain.


* 
**Παραγοντοποίηση:** Υλοποίηση του αλγορίθμου **Pollard's Rho** (με παραλληλισμό) και Trial Division.


* 
**RSA:** Δημιουργία κλειδιών, κρυπτογράφηση και κρυπτανάλυση με την **Επίθεση του Wiener** (για μικρό ).


* 
**Πρωτόκολλα:** Ανάλυση διαφορών **TLS 1.2 vs TLS 1.3** και χρήση PGP/GPG.


* 
**Αρθμητική Υπολοίπων:** Γρήγορη ύψωση σε δύναμη και πολλαπλασιασμός Karatsuba.



**3. Κρυπτογραφία Πλεγμάτων (Lattice-based Cryptography)**
Προχωρημένη κρυπτανάλυση με τεχνικές αναγωγής πλεγμάτων:

* 
**Θεμελιώδη Προβλήματα:** Επίλυση προβλημάτων **SVP** (Shortest Vector Problem) και **CVP** (Closest Vector Problem).


* 
**Αναγωγή Πλέγματος:** Υλοποίηση αλγορίθμων **Gauss-Lagrange** (2D) και **LLL (Lenstra-Lenstra-Lovász)**.


* 
**Κρυπτανάλυση:** Επίθεση σε κρυπτοσυστήματα **Knapsack** με χρήση LLL και μέθοδος **Coppersmith** για εύρεση μικρών ριζών.


* 
**Απαρίθμηση (Enumeration):** Υλοποίηση αλγορίθμων **Babai’s Nearest Plane** και **Schnorr-Euchner**.



**Εργαλεία:**

* Python 3
* SageMath
* LaTeX (για τις αναφορές)
